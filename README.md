# 📚 Mi Biblioteca Personal

Una aplicación web elegante, minimalista y responsiva diseñada para llevar el control y registro físico de tus libros, sus lecturas y ubicaciones de estantes exactas.

## ✨ Características de la aplicación
*   **Gestión Completa:** Añade, edita y elimina libros de tu colección de forma intuitiva.
*   **Persistencia de datos:** Los libros se guardan automáticamente en tu navegador usando `localStorage` (no requiere bases de datos complejas).
*   **Filtros dinámicos:** Busca instantáneamente por título o autores y filtra según su estado de lectura (*Por leer, Leyendo, Leído*).
*   **Diseño Editorial Clásico:** Interfaz optimizada estéticamente con paletas de colores cálidos y tipografías cuidadas (`Playfair Display` y `Lato`).

## 🛠️ Tecnologías utilizadas
*   HTML5 (Estructura semántica)
*   CSS3 (Estilos limpios mediante variables nativas)
*   JavaScript (Lógica nativa ES6 sin dependencias externas ni librerías)

## 🚀 Despliegue en GitHub Pages
Para desplegar este proyecto de manera pública:
1. Sube estos archivos a tu repositorio de GitHub.
2. Dirígete a la pestaña **Settings** (Configuración) de tu repositorio.
3. En el menú lateral izquierdo, haz clic en **Pages**.
4. En la sección *Build and deployment*, selecciona la rama `main` (o `master`) y la carpeta `/root`.
5. Haz clic en **Save** y obtendrás tu enlace público distribuible en pocos minutos.
